# Portfólio - Victor Oliveira Alves

Versão para entrega da atividade CP3 - Portfólio - Front (FIAP).

## Como rodar

```bash
npm install
npm run dev
```

Abra o link fornecido pelo Vite (ex: http://localhost:5173)

## Observações
- Inicia no modo claro por padrão
- Dark mode com persistência via localStorage
- Projeto feito com React + Vite + Tailwind + PostCSS
